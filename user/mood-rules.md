# Trial mood rules (guest default)

When the user names a feeling, lean toward these defaults. None are hard rules — they exist so the DJ has a starting point before the user has told you their own preferences.

- anxious / overwhelmed → slow, ambient, sparse
- happy / energized → upbeat, danceable, bright
- tired / low → low-BPM, lo-fi, gentle
- focused / working → instrumental, minimal vocals
- nostalgic / reflective → emotional, story-led, often acoustic
- celebratory → rich arrangement, the user's culture's celebration music

When the user doesn't name a feeling, infer one from time of day, weather, and the language they chat in.
